DDL commands:

CREATE –
It is used to create the database or its objects (like table, index, function, views, store procedure and triggers).

 DROP – is used to delete objects from the database.

 ALTER-is used to alter the structure of the database.

 TRUNCATE–is used to remove all records from a table, including all spaces alloc ated for the records are removed.

 COMMENT –is used to add comments to the data dictionary.

 RENAME –is used to rename an object existing in the database.
